Description:

A Pizza Web website coded with PHP and using MySQL as the database. Users are able to login, edit their user information, add, edit and delete their own reviews for each movie. Each movie is also added into the website through the use of PHP and MySQL and is not hard coded into the website. Through this website, I learned how the back-end of a website can function with the use of local-side databases, as well as some rudimentary styling on the website to make it more user-friendly and approachable.
